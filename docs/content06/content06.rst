HTTP LBのVIPに対するL7アクセス制御
==================
+---------------+--------------------------------------------+
|送信元         |説明                                        |
+===============+============================================+
|IPv4 Prefix    |指定のネットワークアドレスからのアクセス    |
+---------------+--------------------------------------------+
|BGP ASN        |特定のAS番号からのアクセス                  |
+---------------+--------------------------------------------+
|Country List   |特定の国を指定（実態はグローバルIPアドレス）|
+---------------+--------------------------------------------+
